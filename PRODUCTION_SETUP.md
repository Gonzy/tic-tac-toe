# Настройка Production окружения

## Перед развертыванием

Перед развертыванием в production необходимо заменить следующие значения:

### 1. Файл `infra/envs/backend.prod.env`

Замените `https://your-production-domain.com` на ваш реальный домен фронтенда:

```
ALLOWED_ORIGINS=https://your-real-frontend-domain.com
```

### 2. Файл `infra/envs/frontend.prod.env`

Замените `https://your-production-api-domain.com` на ваш реальный домен бэкенда:

```
REACT_APP_BACKEND_URL=https://your-real-backend-domain.com
```

### 3. Файл `.env`

Убедитесь, что `TELEGRAM_BOT_TOKEN` содержит правильный токен вашего бота.

## Развертывание

Для развертывания в production выполните команду:

```bash
make prod-up
```

Эта команда запустит контейнеры с production конфигурацией.

## Проверка

1. Убедитесь, что фронтенд доступен по вашему домену
2. Проверьте, что API эндпоинты (`/win` и `/lose`) работают корректно
3. Проверьте, что Telegram бот получает сообщения о результатах игр

## Важные моменты

- Убедитесь, что домены настроены правильно и указывают на ваши серверы
- Проверьте, что SSL-сертификаты установлены (если используется HTTPS)
- Убедитесь, что firewall разрешает соединения на необходимых портах
